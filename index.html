SYSTEM DOCUMENTATION

System Configuration Menu

Click the gear icon (⚙) button in the bottom-right corner to open the System Configuration menu. This central tactical dashboard groups all map themes, coordinate grid displays, scale toggles, GIS overlays, and operational telemetry switches into structured categories.

Live GPS Tracking

Toggle 'LIVE GPS TRACKING' inside the System Configuration menu to connect to your device's physical GPS antenna. A pulsing Blue Dot will appear on the map and track your movements in real-time. Use the 'DEPLOY SENSOR AT CURRENT GPS' button inside the menu to drop a marker at your exact physical coordinates.

Tactical Edit Mode & Custom Deployment

Toggle 'TACTICAL EDIT MODE' inside the System Configuration menu to enable map clicking. When active, clicking anywhere on the primary map (or using 'DEPLOY SENSOR AT CURRENT GPS') opens the Deployment Terminal. You can choose a marker category (Radar Blips, Marine Icons, Infrastructure), icon type, and color to deploy custom sensors.

Compass Rose & Coordinate Grid

COMPASS ROSE: Displays a glowing vector dial in the bottom-left corner. The inner needle rotates to indicate true north, while the outer radar dial rotates slowly for orientation context. Toggle it inside the System Configuration menu.
COORDINATE GRID: Overlays latitude and longitude grid lines. Spacing scales dynamically based on zoom to prevent overcrowding. Labels align with the map center crosshair to remain readable and uncluttered.

Advanced Tactical GIS Overlays

Toggle specialized transparent overlays inside the System Configuration menu on top of your base maps:
ROADS & LABELS: Overlays road lines and town labels. Highly useful when viewing raw Satellite base maps.
MTB & OFF-ROAD: Displays mountain biking trails, dirt tracks, and multi-use routes (like the Newfoundland T'Railway) for off-road navigation.
NAUTICAL OVERLAY & HIKING TRAILS: Overlays marine buoys (OpenSeaMap) and official hiking routes.
WEATHER RADAR: Overlays live precipitation radar imagery (sourced in real-time from RainViewer).
TOPO OVERLAY: Displays topographic contour lines to inspect local elevations.
TARGETING RETICLE: Toggles the HUD crosshair overlay centered on the primary viewport.
MAP SCALE: Toggles the map distance scale ruler in the bottom left.

Default Buoys Visibility

Toggle 'DEFAULT BUOYS' inside the System Configuration menu to show or hide the three default preloaded buoys in Deer Lake, letting you isolate only the custom tactical buoys you deploy during operations.

Visual Modes & Map Themes

Select different basemaps from the VISUAL MODE dropdown menu inside the System Configuration menu. Available options include system preference, dark, light, sea (maritime), ESRI satellite, high-resolution satellite, street map, topographic map, and night vision (which applies a green monochrome phosphor filter to the entire application).

Live Cadet Tracking & Uplink

Monitor cadet positions in real-time under the LIVE CADET TRACKING dashboard panel. Remote personnel can access the Cadet Uplink at /transmit.html to register their call sign, select their vehicle/party type, specify party size, and select a custom tactical icon (radar blip, power boat, zodiac, sailboat, ship, truck, or hiker) and color to broadcast their live GPS location. Active SOS states trigger glowing red emergency warnings and audio alarms in the Command Center.

Minimaps & Expandable Views Panel

The right views panel features secondary minimaps to isolate layers (Buoys, EMS, Radar, Forestry). Click the [ EXPAND << ] button at the top of the panel to widen the sidebar to 750px. This snaps the minimaps into a 2x2 grid and increases their heights. Click [ COLLAPSE >> ] to return to narrow view. Use [-] to roll them up, [⛶] to maximize a minimap within the panel (spanning both columns and hiding other panels), and [↗] to pop a minimap out to a full-screen holographic overlay. Click any minimap to instantly fly the main map view to that minimap's center. Click [ < VIEWS ] at the far right edge to collapse or slide the panel completely out of sight.

Intel Feed

The bottom panel is the live system log. It records all system actions, errors, GPS locks, cadet transmissions, and marker deployments for audit tracking.
